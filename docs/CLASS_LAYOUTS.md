# Class, Spec, and Race Layouts

This document details the specific button configurations for Row 3 (Class Panic Row) and the Class sub-page for each class/spec/race combination.

---

## How To Read This Document

- **Row 3 (Class Panic):** 8 buttons, always visible, contains defensives + racial + interrupt + potion
- **Class Page (Rows 1-2):** 16 buttons for spec-specific abilities (position 0,0 is Back button)


# Stream Deck Profile for Paladin (Protection) - Tauren

## Installation

1. Double-click the generated `.streamDeckProfile` file to import it into your Stream Deck software.
2. Select the "WoW" profile in your Stream Deck.

## Keybind Setup

To ensure all buttons work correctly, you must sync your in-game keybinds.

1. Open the file `keybinds.txt` included in this folder.
2. The file contains a list of required keybinds.
3. You can set them manually, or use the provided macro commands if available.

### Important
- "Universal" buttons (Mount, Map) use standard keys (M, Shift-M).
- "Consumables" use standard macros.
- Ensure you have the [AcidWeb Icons](https://github.com/AcidWeb/Clean-Icons-Mechagnome-Edition/) if you are manually editing.

Enjoy your customized layout!

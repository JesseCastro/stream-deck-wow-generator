# StreamDeck Action Bar Setup Guide
## Warrior Protection

After running the keybind installer, you need to place abilities on your action bars.

## Automated Setup (RECOMMENDED):
The install_keybinds.lua script now includes AUTOMATIC action bar placement!
It will attempt to place all your abilities for you.

## Manual Setup (if automation fails):
1. Run the install_keybinds.lua script in WoW
2. Type /reload
3. If some abilities didn't place automatically, drag them from spellbook:

## Tips:
- The script will automatically place spells it can find
- For items (potions, mounts), you need to drag them manually
- Macros need to be created separately and placed manually
- Slots 1-12 are on Action Bar 1 (usually bottom bar)
- Slots 13-24 are on Action Bar 2 (usually right side bar)
- Enable all action bars: ESC → Interface → ActionBars
- If you don't have an ability yet (low level), script will skip it - no errors
